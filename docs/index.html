À propos de moi

  • Qu'est-ce que je fais ?

    Je suis en classe préparatoire et je cherche une alternance pour mon cursus ingénieure en informatique.
    Mais dans mes temps libre je développe mes compétences en développement web.
    J'utilise Udemy, un site qui offre des formations payantes dans tous les domaines.

  • Quelles outils j'utilise ?

    J'utilise le javascript native pour le scripting.
    La plupart du temps avec des class ES6, localStorage, IntersectionObserver API pour les scroll events et Async/Await pour faire appel aux API.
    Pour styliser, j'utilise SCSS, c'est plus pratique.
    Car avec SCSS je peux créer des composants réutilisable.

  • Et après ?

    Je veux apprendre les bases du développement back-end avec Node.js, express, mongoose. Ensuite, m'attaquer à un framework javascript comme ReactJS.
